### 图形界面应用程序：

- 基于消息驱动模型的可执行程序
- 程序的执行依赖于用户的交互过程
- 程序的执行过程中实时响应用户操作
- 一般情况下程序执行后不会主动退出

GUI的运行模式：

main -> 创建主窗口 -> 进入消息循环 -> 创建主窗口中的元素 -> 显示主窗口

GUI的消息处理模型：

用户操作 -- 到操作系统--- > 系统内核 ——> 系统消息 -----> 消息处理函数
